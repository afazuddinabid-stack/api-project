১. var, let, এবং const-এর মধ্যে পার্থক্য


JavaScript-এ ভ্যারিয়েবল ডিক্লেয়ার করার জন্য এই তিনটি কীওয়ার্ড ব্যবহার করা হয়।

var: এটি পুরনো পদ্ধতি। এর Function Scope আছে। একই নামে বারবার ভ্যারিয়েবল ডিক্লেয়ার করা যায়, যা অনেক সময় কোডে বাগ (bug) তৈরি করে।

let: এটি আধুনিক পদ্ধতি। একই ব্লকে একই নামে দ্বিতীয়বার ভ্যারিয়েবল ডিক্লেয়ার করা যায় না, তবে মান পরিবর্তন করা যায়।

const: এটি let-এর মতোই, কিন্তু এর মান একবার সেট করলে আর পরিবর্তন করা যায় না।

২. Spread Operator (...) 


Spread operator-কে তিনটি ডট (...) দিয়ে প্রকাশ করা হয়। এটি মূলত কোনো Array বা Object-এর উপাদানগুলোকে আলাদা আলাদা করে ছড়িয়ে দিতে সাহায্য করে।

ব্যবহার: একটি Array-এর সব উপাদান অন্য একটি Array-তে কপি করতে বা একাধিক Array জোড়া

৩. map(), filter(), এবং forEach()-এর পার্থক্য

এই তিনটি মেথডই Array নিয়ে কাজ করার জন্য ব্যবহৃত হয়।
forEach()Array-এর প্রতিটি উপাদানের ওপর কোনো কাজ চালায়।কিছুই রিটার্ন করে না
 (undefined)map()প্রতিটি উপাদানকে পরিবর্তন করে নতুন একটি Array তৈরি করে।নতুন একটি Arrayfilter()শর্ত সাপেক্ষে নির্দিষ্ট কিছু উপাদান নিয়ে নতুন Array তৈরি করে।ফিল্টার করা নতুন একটি Array

৪. Arrow Function 

Arrow function হলো ফাংশন লেখার একটি  আধুনিক পদ্ধতি। এটি দেখতে  এইরকম: () => { }

এটি কোডকে ছোট  রাখে।

সাধারণ ফাংশনের মতো এতে function কীওয়ার্ড লিখতে হয় না।

৫. Template Literals 

Template literals হলো স্ট্রিং (String) লেখার একটি  পদ্ধতি | যেখানে কোটেশনের পরিবর্তে Backticks (`) ব্যবহার করা হয়।
 ${ } ব্যবহার করে স্ট্রিংয়ের ভেতরে সরাসরি ভ্যারিয়েবল বসানো যায়।

মাল্টি-লাইন: ব্যাকটিক ব্যবহার করলে খুব সহজেই মাল্টি-লাইন স্ট্রিং লেখা যায়, যা সাধারণ সিঙ্গেল বা ডাবল কোটে সম্ভব ছিল না।
